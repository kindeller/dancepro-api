# Brand Style

## Purpose

Document stable visual guidance for DancePro V2 admin and client-facing
interfaces.

## Colour

The primary DancePro brand colour is:

```text
#0AA0DB
```

Use this blue for primary actions, active navigation states, focus states, and
brand accents.

Prefer a clean black, white, and cool off-white base around the brand colour:

- Dark navigation/base: `#101820`
- Primary text: `#141719`
- Muted text: `#66737a`
- Page background: `#f7fbfd`
- Borders: `#d7e4ea`
- Soft blue surface: `#eaf8fd`

## Interface Direction

DancePro interfaces should feel clean, practical, and image-media focused.

Prefer:

- restrained layouts with clear hierarchy;
- mostly square controls and cards, currently `4px` radius;
- high contrast dark navigation with blue active states;
- simple uppercase navigation labels where appropriate;
- blue primary buttons with darker blue hover states.

Avoid:

- unrelated accent colours for primary actions;
- highly rounded pill-heavy styling;
- decorative gradients that compete with photography or video content.
